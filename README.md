# Roll of Fate

## Description

Roll of Fate is a Java Swing-based Dungeons & Dragons-inspired adventure game developed using Object-Oriented Programming principles. Players create a hero, explore dangerous dungeon rooms, battle monsters, collect gold and experience, purchase upgrades and potions, and ultimately face the final boss, the Ancient Shadow Dragon.

The game uses a dice-based combat system where D20 rolls determine attack success and various dice types determine damage. Special gameplay rules such as Adaptive AI, Boss Rage Mode, Double Dice Damage, Traps, and Permadeath can be enabled to increase challenge and variety.

## Features

* Hero classes: Warrior, Mage, Archer, and Rogue
* Dice-based combat system
* Turn-based battles
* Random monster encounters
* Final boss battle
* Inventory and potion system
* Weapon and armor upgrades
* Dungeon shop
* Adaptive enemy AI
* Boss Rage Mode
* Trap system
* Status effects and temporary buffs
* GUI built with Java Swing
* Multiple optional game rules
* Experience, leveling, and gold rewards

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming Concepts

* Encapsulation

* Abstraction

* Interfaces
* Exception Handling
* Information Hiding

## Win Condition

More of a Survival mode game.

## Lose Condition

If the hero's health reaches zero, the game ends. If Permadeath is enabled, the run immediately ends upon death.
